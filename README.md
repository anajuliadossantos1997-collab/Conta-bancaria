Esse é um projeto em linguegem Python que simula operações bancárias básicas como: Depósito, Saque, Transferência e visualização de extrato.
Funcionalidades: 
Depósito; permite adicionar valores ao saldo.
Saque; permite retirar valores, respeitando o saldo disponível.
Extrato; Lista todas as movimentações realizadas.

Para rodar o sistema, Você precisa ter o Python instalado. Basta abrir o terminal na pasta e digitar python Conta-bancario.py
